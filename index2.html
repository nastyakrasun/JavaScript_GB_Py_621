Урок 2. Основные операторы JavaScript

  1. Почему данный код даёт именно такие результаты?

    var a = 1, b = 1, c, d;

    создали переменные a, b, c, d. a, b - с заданными начальными значениями, c, d - просто создали

    c = ++a; alert(c); // 2

    присвоили переменной с значение увеличенной на 1 переменной a

    d = b++; alert(d); // 1

    присвоили переменной d значение переменной b. после чего переменную b увеличели на 1

    c = (2+ ++a); alert(c); // 5

    переменная а сейчас равна 2. увеличиваем её на 1 и прибавляем к цифре 2

    d = (2+ b++); alert(d); // 4

    переменная b сейчас равна 2. прибавляем её к цифре 2, затем увеличиваем на 1

    alert(a); // 3

    выводим переменную а, которая после всех операций равна 3

    alert(b); // 3

    аналогично выводим переменную b

  2. Чему будет равен x в примере ниже? var a = 2; var x = 1 + (a *= 2);

    1 + 2*2 = 5
